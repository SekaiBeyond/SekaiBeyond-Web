# SekaiBeyond Web

Official website for Sekai Beyond built with React and TypeScript.

## Getting Started

### Installation

```bash
git clone https://github.com/SekaiBeyond/SekaiBeyond-Web.git
cd SekaiBeyond-Web
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```
